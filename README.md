

###简介
通过behavior实现的类似京东淘宝商品详情页面的上拉下拉页面的切换效果。

![DoubleScrollView](demo.gif)

	
### 用法
1. 依赖doublescrollview library模块或者直接拷贝PageBehavior和res文件下的文件到项目中。
2. 为coordinatelayout中的作为第二页的组件添加 behavior属性为pagehebavior即可。
3. 具体设置见demo。
